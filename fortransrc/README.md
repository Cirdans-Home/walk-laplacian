# Fortran Sources

Here is the code needed to run the tests on the solution of the linear systems that are used to define the NBT Laplacian based on Katz centrality.

Once you have installed PSCToolkit in the provided install folder, you just need to do `make` in this directory to compile the examples that will then be moved to the runs folder at the top level.

The code can be run even if you have not configured PSCToolkit to work with the GPU.
